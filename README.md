# alloview

`alloview` is a `NextJS` web application for viewing allometric models and their
publications. It is part of the larger `allometric` constellation of software.
The objective of `alloview` is to reduce the level of effort needed to interact
with, search, and find allometric models used in forest resource analysis.

This repository contains the source code for `alloview`, and the main
application can be viewed [here]().
